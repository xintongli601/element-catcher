# Element Catcher

Element Catcher is a planned Chrome extension for selecting, inspecting, and working with elements on a web page. This repository currently contains the product and development documentation foundation only; extension implementation has intentionally not started yet.

The initial project setup is intentionally minimal. Before any source code, build tooling, frameworks, or dependencies are added, the Product Requirements Document and development brief should be finalized in `docs/` so implementation can follow the agreed scope.
